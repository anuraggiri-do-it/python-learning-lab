## ⚡ Quick Start

```bash
git clone https://github.com/anuraggiri-do-it/python-learning-lab.git
cd python-learning-lab
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```
